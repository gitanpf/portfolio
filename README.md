# 💻 Meu Portfólio Pessoal

Este é o repositório do meu site de portfólio pessoal, desenvolvido para centralizar meus projetos, habilidades e trajetórias na Engenharia Front-End. O objetivo principal foi criar uma interface limpa, moderna e totalmente responsiva utilizando apenas tecnologias fundamentais da web.

---

### 🚀 Tecnologias Utilizadas

Para este projeto, foquei na semântica do HTML e no poder de estilização do CSS moderno:

- **HTML5:** Estruturação semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização personalizada, utilizando Flexbox e CSS Grid para o layout.
- **Mobile-First:** Metodologia aplicada para garantir que o site funcione perfeitamente em qualquer dispositivo.

---

### 🛠️ Funcionalidades e Diferenciais

- **Design Responsivo:** Adaptado para mobile, tablet e desktop.
- **Seção de Projetos:** Galeria exibindo meus principais trabalhos (React, Angular, etc).
- **Formulário de Contato:** Layout funcional para recepção de mensagens.
- **Performance:** Carregamento rápido devido ao uso de código nativo, sem bibliotecas pesadas.
